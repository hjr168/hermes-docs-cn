# Hermes Agent v0.10.0 — Tool Gateway 发布

**发布日期：** 2026 年 4 月 16 日

> Tool Gateway 版本 — 付费 Nous Portal 订阅用户现在可以通过现有订阅直接使用网页搜索、AI 生图、语音合成和浏览器自动化，无需额外配置任何 API Key。

---

## ✨ 亮点功能

### Nous Tool Gateway

付费 [Nous Portal](https://portal.nousresearch.com) 订阅用户现在可以自动获得以下工具的使用权限：

| 工具 | 服务商 | 说明 |
|------|--------|------|
| **网页搜索** | Firecrawl | 搜索互联网获取实时信息 |
| **AI 生图** | FAL / FLUX 2 Pro | 根据文字描述生成图片 |
| **语音合成** | OpenAI TTS | 将文字转换为自然语音 |
| **浏览器自动化** | Browser Use | 自动操控浏览器完成网页任务 |

**零配置上手：** 不再需要单独申请 API Key，只需运行 `hermes model`，选择 Nous Portal，然后挑选要启用的工具即可。

**灵活可控：** 每个工具都可通过 `use_gateway` 配置独立开关，`hermes tools` 和 `hermes status` 命令完整支持网关工具的管理与状态查看。当同时存在直连 API Key 和网关时，运行时会优先使用网关。

**替代旧方案：** 新的订阅检测机制取代了之前的隐藏环境变量 `HERMES_ENABLE_NOUS_MANAGED_TOOLS`，更加清晰直观。

相关 PR：[#11206](https://github.com/NousResearch/hermes-agent/pull/11206)，文档更新：[#11208](https://github.com/NousResearch/hermes-agent/pull/11208)
