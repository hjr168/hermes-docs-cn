# Hermes Agent v0.9.0 — 无处不在的 Hermes

**发布日期：** 2026 年 4 月 13 日
**自 v0.8.0 以来：** 487 commits · 269 合并 PR · 167 已解决 Issue · 493 文件变更 · 63,281 行新增 · 24 位贡献者

> "Everywhere" 版本 — Hermes 登陆手机端（Termux/Android），新增 iMessage 和微信支持，上线 Fast Mode，引入后台进程监控，发布本地 Web 仪表盘，并完成了史上最深度的安全加固。

---

## ✨ 亮点功能

### 🖥️ 本地 Web 仪表盘

全新的浏览器端管理界面，无需触碰配置文件或终端即可：
- 配置 Agent 设置
- 监控会话状态
- 浏览和管理 Skill
- 管理网关连接

这是体验 Hermes 最简单的方式。

### ⚡ Fast Mode（`/fast`）

为 OpenAI 和 Anthropic 模型提供优先处理通道。输入 `/fast` 即可通过优先队列显著降低延迟，支持 GPT-5.4、Codex、Claude 等模型。

### 💬 iMessage（BlueBubbles）

完整的 iMessage 集成，将 Hermes 带入 Apple 消息生态。支持自动 Webhook 注册、设置向导和崩溃恢复。

### 🟢 微信 & 企业微信

- **微信**：通过 iLink Bot API 实现原生支持，支持流式输出、媒体上传、Markdown 链接处理
- **企业微信回调模式**：适配自建企业应用，支持原子化状态持久化

Hermes 现已完整覆盖中文消息生态。

### 📱 Termux / Android 支持

在 Android 手机上通过 Termux 原生运行 Hermes，适配安装路径、移动端 TUI 优化、语音后端支持，`/image` 命令可在设备端运行。

### 🔔 后台进程监控（`watch_patterns`）

设置输出模式匹配规则，匹配时实时通知。可监控错误、等待特定事件（如 "listening on port"）或追踪构建日志——无需轮询。

### 🤖 原生 xAI & 小米 MiMo 提供商

xAI（Grok）和小米 MiMo 成为一级提供商，支持直连 API 访问、模型目录和设置向导集成。

### 🧩 可插拔上下文引擎

通过 `hermes plugins` 切换上下文管理引擎——自定义过滤、摘要或注入领域特定上下文。

### 🌐 统一代理支持

SOCKS 代理、`DISCORD_PROXY` 和系统代理自动检测，跨所有网关平台生效。企业防火墙后的 Hermes 开箱即用。

### 🔒 全面安全加固

路径遍历防护、Shell 注入中和、SSRF 重定向防护、Twilio Webhook 签名验证（修复 SMS RCE）、API 服务器认证强制执行、Git 参数注入防护等。

### 💾 `hermes backup` & `hermes import`

完整的配置备份与恢复——迁移到新机器或在重大变更前创建快照。

### 🏗️ 16 个支持平台

随着 BlueBubbles（iMessage）和微信的加入，Hermes 现已支持 16 个消息平台：Telegram、Discord、Slack、WhatsApp、Signal、Matrix、Email、SMS、钉钉、飞书、企业微信、Mattermost、Home Assistant、Webhooks、iMessage、微信。
