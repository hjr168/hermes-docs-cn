# Hermes Agent v0.13.0 — Tenacity 版本

**发布日期：** 2026 年 5 月 7 日
**自 v0.12.0 以来：** 864 commits · 588 合并 PR · 829 文件变更 · 128,366 行新增 · 282 个 issues 关闭（13 P0, 36 P1）· 295 位社区贡献者

> Tenacity 版本 — Hermes Agent 有始有终。Kanban 作为持久多 Agent 看板正式发布（心跳、回收、僵尸检测、自动阻塞、每任务重试、幻觉恢复）。`/goal` 让 Agent 跨轮次锁定目标（Ralph loop）。Checkpoints v2 重写状态持久化。Gateway 自动恢复中断会话。Cron 新增 `no_agent` 看门狗模式。一波安全更新关闭 8 个 P0 漏洞。Google Chat 成为第 20 个消息平台。Provider 变为可插拔表面。7 个 i18n locale 正式发布。

---

## ✨ 亮点功能

### 📋 Multi-Agent Kanban — 交给 AI 团队，它们真的能完成

启动一个持久看板，丢任务上去，让多个 Hermes Worker 去pick up、交接和完成。心跳、回收、僵尸检测、重试预算、幻觉门控保证团队诚实。一个安装，多个看板。

### 🎯 `/goal` — Agent 不会忘记你让它做什么

锁定 Agent 到一个目标，它跨轮次保持专注。Ralph loop 作为一级原语。

### 🎬 给它看视频

新增 `video_analyze` 工具，在 Gemini 及兼容多模态模型上进行原生视频理解。

### 🗣️ 克隆你的声音

xAI Custom Voices 作为 TTS Provider，支持声音克隆。

### 🌍 Hermes 说你的语言

静态 Gateway + CLI 消息翻译至 7 个 locale：**中文**、日语、德语、西班牙语、法语、乌克兰语、土耳其语。文档站新增中文 (zh-Hans) locale。

### 💬 Google Chat — 第 20 个消息平台

通用平台插件钩子表面——第三方适配器无需触碰核心即可接入（IRC 和 Teams 已迁移）。

### 🔄 会话在重启后存活

Gateway 中途重启、`/update` 重启、源文件重载——对话在 Gateway 恢复时自动继续。

### 🔒 安全更新 — 关闭 8 个 P0

- 脱敏默认开启
- Discord 角色允许列表改为 guild-scoped（CVSS 8.1 跨 guild DM 绕过已关闭）
- WhatsApp 默认拒绝陌生人
- TOCTOU 窗口在 `auth.json` 和 MCP OAuth 中关闭
- 浏览器强制云元数据 SSRF 地板
- Cron prompt 注入扫描组装技能内容
- `hermes debug share` 上传时脱敏

### 💾 Checkpoints v2

状态持久化全面重写。真正的裁剪、磁盘保护、再无孤立 shadow repos。

### ✍️ Agent 检查自己的代码

`write_file` + `patch` 后写增量 lint。Python、JSON、YAML、TOML。语法错误立即浮现而非延后到运行时。

### ⏰ `no_agent` Cron 模式 — 纯脚本看门狗

Cron 任务可以完全跳过 Agent，只运行脚本。空 stdout 静默，非空 stdout 原样投递。

### 🛡️ Platform 允许列表无处不在

`allowed_channels` / `allowed_chats` / `allowed_rooms` 配置横跨 Slack、Telegram、Mattermost、Matrix、DingTalk。

### 🔌 Provider 变为插件

`ProviderProfile` ABC + `plugins/model-providers/`。第三方 Provider 无需触碰核心即可接入。

### 📡 MCP 升级

SSE 传输 + OAuth 转发、粘管重试、图片结果作为 MEDIA 标签浮现、长生命周期等待保活。

### 🧹 Curator 新增子命令

`hermes curator archive`、`prune`、`list-archived`。手动 `hermes curator run` 改为同步——立即看到结果。

### 🎨 TUI 升级

`/model` 选择器与 `hermes model` 一致、内联认证、可折叠启动横幅、状态栏显示上下文压缩计数。

### 📊 Dashboard 升级

插件页面（管理、启用/禁用、认证状态）、Profiles 管理页面、可排序分析表、反向代理支持、`default-large` 18px 主题。

### 🔍 SearXNG + 拆分 Web 工具

SearXNG 作为原生搜索后端发货；Web 工具现在支持按能力选择不同后端（搜索 vs 提取 vs 浏览）。

### 📝 其他重要更新

- **API server — 每会话长期记忆** — `X-Hermes-Session-Key` header
- **`[[as_document]]` — 技能媒体路由指令**
- **`transform_llm_output` 插件钩子**
- **Nous OAuth 跨 Profile 持久化**
- **QQBot 原生审批键盘**
- **6 个新可选技能** — Shopify、here.now、shop-app、Anthropic 金融套装、kanban-video-orchestrator、searxng-search
- **新模型** — `deepseek/deepseek-v4-pro`、`x-ai/grok-4.3`、`openrouter/owl-alpha`（免费）等
