# Hermes Agent v0.12.0 — Curator 版本

**发布日期：** 2026 年 4 月 30 日
**自 v0.11.0 以来：** 1,096 commits · 550 合并 PR · 1,270 文件变更 · 217,776 行新增 · 213 位社区贡献者

> Curator 版本 — Hermes Agent 现在自主维护自己。后台 Curator 自动对技能库进行评级、裁剪和整合。自改进循环全面升级。4 个新推理提供商、第 18 个消息平台（腾讯元宝）、第 19 个 via Teams 插件、原生 Spotify + Google Meet 集成、ComfyUI 和 TouchDesigner-MCP 从可选升级为内置、TUI 冷启动速度提升约 57%。

---

## ✨ 亮点功能

### 🧹 Autonomous Curator — 自动技能维护

`hermes curator` 作为后台 Agent 在网关的 cron  ticker 上运行（默认 7 天周期）。它对技能库进行评级、整合相关技能、裁剪废弃技能，并将每次运行报告写入 `logs/curator/run.json` + `REPORT.md`。归档技能通过模型 + 启发式分类为"整合"vs"裁剪"。防御深层保护内置/Hub 技能不被修改。统一在 `auxiliary.curator` 下——在 `hermes model` 中选择 Curator 的模型，从仪表盘管理。`hermes curator status` 按使用量对技能排名。

### 🧠 自改进循环 — 全面升级

后台复盘分支（Hermes 自改进的核心：每轮后决定保存或更新哪些记忆/技能）现已 Class-first（基于评分表而非自由形式）、主动更新偏向（偏好刚加载的技能）、正确处理 `references/`/`templates/` 子文件、正确继承父级运行配置（Provider、模型、凭据实际传播）。仅限 memory + skills 工具集防止蔓延。记忆提供者正常关闭。前一轮工具消息从摘要中排除。

### 🧩 技能集成 — 大幅扩展

- **ComfyUI v5** — 官方 CLI + REST + 硬件门控本地安装，从可选升级为**内置默认**
- **TouchDesigner-MCP** 内置默认，新增 GLSL、Post-FX、音频、几何和 9 份新参考文档
- **Humanizer** 技能 — 剥离 AI 用语的文本清理器
- **claude-design** HTML 制品技能 + design-md (Google DESIGN.md 规范) + direct-URL 技能安装 + `/reload-skills` 斜杠命令

### 🖥️ LM Studio — 一级 Provider

从 custom-endpoint 别名升级为完整原生 Provider：专用认证、`hermes doctor` 检查、reasoning 传输、实时 `/models` 列表。

### 🔮 4 个新推理 Provider

- **GMI Cloud** — 一级原生支持
- **Azure AI Foundry** — 自动检测
- **MiniMax OAuth** — PKCE 浏览器流程
- **Tencent Tokenhub**

### 💬 腾讯元宝 (Yuanbao) — 第 18 个消息平台

原生网关适配器，支持文本和媒体投递。

### 📋 Microsoft Teams — 第 19 个消息平台（插件形式）

网关现为插件主机。可插拔消息适配器置于核心之外，Microsoft Teams 作为首个插件平台发货。

### 🎵 Spotify — 原生工具 + 内置技能 + 向导

7 个工具（播放、搜索、队列、播放列表、设备） behind PKCE OAuth，交互式设置向导，内置技能。

### 🏛️ Google Meet 插件

加入会议、转录、说话、跟进。Realtime OpenAI 传输 + Node bot server，完整 pipeline 作为插件打包。

### ⚡ TUI 冷启动加速 ~57%

通过懒加载 Agent 初始化、懒导入 OpenAI/Anthropic/Firecrawl、mtime 缓存 `load_config()`、记忆化 `get_tool_definitions()` 等优化。

### 📝 其他重要更新

- **`hermes -z` 单次模式** + **`hermes update --check`** — 非交互式单次执行
- **模型仪表盘标签页** — 丰富的每模型分析，仪表盘内切换主模型和辅助模型
- **远程模型目录清单** — OpenRouter + Nous Portal 模型目录从远程清单拉取
- **原生多模态图像路由** — 图像根据模型实际视觉能力而非 Provider 默认值路由
- **Gateway 媒体对等** — Telegram、Discord、Slack、Mattermost、Email、Signal 原生多图发送
- **Secret 脱敏默认关闭** — 防止长期存在的补丁损坏问题
- **可配置 Prompt Cache TTL** — `prompt_caching.cache_ttl`
