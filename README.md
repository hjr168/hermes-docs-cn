# Hermes Agent 中文文档

> 非官方社区翻译项目

本站是 [Hermes Agent](https://github.com/NousResearch/hermes-agent) 文档的中文翻译版本，基于 Docusaurus 构建。

## 免责声明

- 本项目为**非官方**社区翻译，与 Nous Research 无关
- 翻译内容仅供参考，请以[英文原版文档](https://github.com/NousResearch/hermes-agent/tree/main/website/docs)为准
- 如发现翻译错误，欢迎提交 Issue 或 Pull Request

## 本地开发

```bash
npm install
npm start        # 启动开发服务器 (http://localhost:3000)
npm run build    # 构建生产版本
npm run serve    # 本地预览构建结果
```

## 贡献

欢迎贡献翻译！请确保：

- 保持 Markdown 格式和 frontmatter 不变
- 技术术语保留英文原文，首次出现时附中文注释
- 代码块中的注释也翻译为中文
